:::tip 是否不为 null | undefined | false | NaN
```js
VerifyHelper.isTrue(val);
```
:::

## 版本

`v4.3.1+`

## 参数

- `val` （String）：需要判断的内容

## 返回

-（Boolean）：返回真或假

## 例子

```js
VerifyHelper.isTrue(1);
// => true
```
