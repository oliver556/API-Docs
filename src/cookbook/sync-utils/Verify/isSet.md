:::tip 是否 Set 对象
```js
VerifyHelper.isSet(val);
```
:::

## 版本

`v4.3.1+`

## 参数

- `val` （String）：需要判断的内容

## 返回

-（Boolean）：返回真或假

## 例子

```js
VerifyHelper.isSet(new Set());
// => true
```
