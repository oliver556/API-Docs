:::tip 删除 session 缓存
```js
EnvelopHelper.removeSessionStorage(key);
```
:::

## 版本

`v4.3.1+`

## 参数

- `key` （String）：键名

## 返回

无

## 例子

```js
EnvelopHelper.removeSessionStorage('key');
// => 返回key对应的键名
```
