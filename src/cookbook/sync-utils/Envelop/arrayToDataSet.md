:::tip Array 转 DataSet
```js
EnvelopHelper.arrayToDataSet(array);
```
:::

## 版本

`v4.3.1+`

## 参数

- `array` （Array）：需要转 dataset 的数组

## 返回

-（Array）：转换后的dataset

## 例子

```js
EnvelopHelper.arrayToDataSet(array);
// => dataset
```
