:::tip 设置 session 缓存
```js
EnvelopHelper.setSessionStorage(key, envelop);
```
:::

## 版本

`v4.3.1+`

## 参数

- `key` （String）：键名
- `envelop`（Envelop）：键值

## 返回

无

## 例子

```js
EnvelopHelper.setSessionStorage('key', envelop);
```
