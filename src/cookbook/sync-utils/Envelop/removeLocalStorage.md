:::tip 删除 local 缓存
```js
nvelopHelper.removeLocalStorage(key);
```
:::

## 版本

`v4.3.1+`

## 参数

- `key` （String）：键名

## 返回

无

## 例子

```js
EnvelopHelper.removeLocalStorage('key');
// => 返回 key 对应的键名
```
