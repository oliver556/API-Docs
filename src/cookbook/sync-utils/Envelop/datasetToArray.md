:::tip DataSet 转 Array
```js
EnvelopHelper.datasetToArray(dataset);
```
:::

## 版本

`v4.3.1+`

## 参数

- `dataset` （Array）：信封 dataset 对象

## 返回

-（Array）：转换后的数组对象

## 例子

```js
EnvelopHelper.datasetToArray(dataset);
// => array
```
