:::tip 设置 local 缓存
```js
EnvelopHelper.setLocalStorage(key, envelop);
```
:::

## 版本

`v4.3.1+`

## 参数

- `key` （String）：键名
- `envelop`（Envelop）：键值

## 返回

无

## 例子

```js
EnvelopHelper.setLocalStorage('key', envelop);
```
